# Defina uma porta alternativa para o SMTP no WHM

1. Efetue login no *WHM*.
2. Clique em *Gerenciador de Serviços*.
3. Configure a porta após *Executar outra cópia do Exim na porta*.
4. Clique em *Salvar*.
