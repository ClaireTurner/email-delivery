# Alternativen Port für SMTP in WHM definieren

1. Melden Sie sich bei *WHM* an.
2. Klicken Sie auf *Service Manager*.
3. Geben Sie den Port nach *Weitere Exim-Kopie ausführen an Port* an.
4. Klicken Sie auf *Speichern*.
